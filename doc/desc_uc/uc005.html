UC005 - Listar Categoria

Objetivo

O sistema informará quais os tipos de produtos possuem.

Atores

  • Usuário

Pré-condições

  • O ator deve está logado no sistema.
  • O ator deverá está tela de exibir categorias.

Pós-condições

  • As categorias devem ser exibidas ao usuário.
