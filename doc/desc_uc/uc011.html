UC011 - Calcular Total

Objetivo

O sistema deve mostrar o preço total dos itens daquele pedido.

Atores

  • Usuário

Pré-condições

  • O ator deve estar logado no sistema.
  • O ator deve selecionar todos os produtor desejados.

Pós-condições

  • O preço total dos itens deve ter sido mostrado ao usuário.

Fluxo Principal
