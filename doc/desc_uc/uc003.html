UC003 - Realizar Pedido

Objetivo

O ator deve poder realizar um pedido.

Atores

  • Usuário

Pré-condições

  • O ator deve estar logado no sistema.
  • O ator deve ter itens no carrinho.
  • O método de pagamento do cliente deve ser válido

Pós-condições

  • O pedido deve ser realizado e o vendedor notificado.
