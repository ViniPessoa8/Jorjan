UC010 - Remover item do carrinho

Objetivo

O usuário poderá remover itens do carrinho.

Atores

  • Usuário

Pré-condições

  • O ator deve estar logado no sistema.
  • O ator deve ter itens no carrinho.
  • O ator deve estar na tela de carrinho.

Pós-condições

  • O itens indesejados devem estar removidos do carrinho.
