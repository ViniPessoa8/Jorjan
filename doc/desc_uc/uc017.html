UC017 - Confirmar Entrega

Objetivo

O usuário que vendeu, deve confirmar que entregou o produto..

Atores

  • Usuário

Pré-condições

  • O ator deve está logado no sistema.
  • O ator deve estar com o pedido em andamento.
  • O ator deve ter entregado o produto

Pós-condições

  • O status do pedido muda para "entregue" e a compra é finalizada.
