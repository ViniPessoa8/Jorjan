UC013 - Confirmar Pedido

Objetivo

O ator deve poder confirmar um pedido.

Atores

  • Vendedor

Pré-condições

  • O ator deve estar logado no sistema.
  • O ator deve ter produtos cadastrados.
  • O ator deve estar disponível para receber pedidos.
  • O ator deve ter recebido um pedido.

Pós-condições

  • O pedido foi confirmado e aguardará pela entrega.
