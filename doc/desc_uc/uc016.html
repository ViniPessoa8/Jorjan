UC016 - Notificar Vendedor

Objetivo

O sistema deve enviar uma notifição ao vendedor, informando uma nova venda.

Atores

  • Vendedor

Pré-condições

  • O ator deve estar logado no sistema.
  • O ator deve ter produtos cadastrados.
  • O ator deve estar disponível para receber pedidos.

Pós-condições

  • A notificação deve ser recebida pelo ator.
