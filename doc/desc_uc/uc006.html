UC006 - Listar Vendedores

Objetivo

O sistema informará quais vendedores estão cadatrados

Atores

  • Usuário

Pré-condições

  • O ator deve está logado no sistema.
  • O ator deverá está tela de exibir vendedores.

Pós-condições

  • Os vendedores disponíveis devem ser exibidos ao usuário.

Fluxo Principal
