UC012 - Finalizar Compra

Objetivo

O usuário deve poder finalizar a compra.

Atores

  • Usuário

Pré-condições

  • O ator deve estar logado no sistema.
  • O ator deve ter entregado o produto.
  • O ator deve estar na tela de pedidos.

Pós-condições

  • A compra deve estar encerrada.

Fluxo Principal
