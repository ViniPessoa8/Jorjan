UC012 - Pesquisar

Objetivo

O usuário deve poder realizar pesquisas de categorias, produtos e vendedores.

Atores

  • Usuário

Pré-condições

  • O ator deve estar logado no sistema.
  • A barra de pesquisa deve estar visível.

Pós-condições

  • Uma lista com os resultados para a pesquisa deverão ser exibidos na tela.
