UC009 - Adicionar item ao carrinho

Objetivo

O usuário deve poder adicionar o itens desejados ao carrinho

Atores

  • Usuário

Pré-condições

  • O ator deve estar logado no sistema.
  • O ator deve estar na tela de selecionar produtos.

Pós-condições

  • Os itens do pedido devem estar todos no carrinho.
