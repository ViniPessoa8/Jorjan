UC021 - Denunciar Usuário

Objetivo

O usuário pode denunciar um outro usuário com perfil impróprio.

Atores

  • Usuário

Pré-condições

  • O ator deve está logado no sistema.
  • O ator deve estar no perfil de um vendedor.

Pós-condições

  • A denuncia deve ser enviada ao sistema.
