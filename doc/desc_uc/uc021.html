UC021 - Editar Perfil

Objetivo

O usuário pode editar as informações de seu perfil.

Atores

  • Usuário

Pré-condições

  • O ator deve está logado no sistema.
  • O ator deve estar na tela de perfil.

Pós-condições

  • As informações no perfil do usuário deverão estar alteradas.
