UC007 - Listar Produtos

Objetivo

O ator deve poder se cadastrar no sistema.

Atores

  • Usuário

Pré-condições

  • O ator deve está logado no sistema.
  • O ator deve estar na tela de exibir produtos.

Pós-condições

  • Os produtos devem ser exibidos ao usuário ao usuário.
