UC007 - Selecionar Vendedor

Objetivo

O ator deve poder selecionar um vendedor para ver seus produtos.

Atores

  • Usuário

Pré-condições

  • O ator deve está logado no sistema.
  • O ator deve estar na tela de exibir vendedores.

Pós-condições

  • Os produtos devem ser exibidos ao usuário.

Fluxo Principal
