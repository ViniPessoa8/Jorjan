# Backend - Flask

- Requirements:
    - Anaconda:
        - https://docs.anaconda.com/anaconda/install/
    - Pip:
        - ```apt install python3-pip```

- Setup:
    - ```source bin/activate```
    - ```pip install -r requiremets.tx```

- Running*:
    - ```flask run```


*Obs.: Sempre rodar com o venv ativado (```source bin/activate```)
